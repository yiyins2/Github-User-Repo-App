# Github User Repo Info App

This project create four Github-related screens for users to navigate: 

- Profile Screen: 
  - Avatar
  - Name
  - Github username
  - Bio
  - Website 
  - Email 
  - Profile creation date 
- Repositories Screen: 
  - Repository's name
  - Owner
  - Description
- Follower Screen: 
  - Avatar
  - Name 
  - Username
- Following Screen: 
  - Avatar
  - Name 
  - UsernameR

The project was developed in React Native. 
To run the app, run `expo start`.
